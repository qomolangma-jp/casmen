セルフ面接のやり方

カメラ・マイクをON
画面に出る許可ポップアップで「OK」をタップください。
録画ボタンスタート
セルフ面接スタートボタンをタップしてから、3秒後に質問が始まります。
質問は12問・約2分
1問につき約10秒。テンポよく表示される質問に、あなたのペースで答えてください。
やり直しは1回だけOK
「失敗した！」と思ったら、
もう一度だけ録画できます。
最後に確認して送信
確認画面で内容を見て
「送信」を押せば完了です。
