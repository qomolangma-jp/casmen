システムメンテナンスのお知らせ

2025/07/02/ 02:50

いつも【もえなび！・カフェるん】をご利用いただきまして誠にありがとうございます。

この度、クレジットカード決済のセキュリティ強化のため、下記の通りメンテナンスを実施いたします。

■メンテナンス日時
2025年2月24日(月) 13:00 ～ 14:00

■メンテナンス理由
経済産業省の「クレジットカード・セキュリティガイドライン」に基づき、オンライン決済における3Dセキュア（本人認証サービス）の導入が義務化されます。
これに対応を行い、お客様に安心してご利用いただくため、システムの改修を行います。

■メンテナンス期間中の影響
・クレジットカード決済のご利用停止
・自動決済登録の停止
※店舗様の管理画面、広告購入は通常通りご利用いただけます。
